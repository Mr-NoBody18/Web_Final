  3. API anahtarı olmadan da site hakkında bilgi verebilir

• "Haberler nerede?"

🔑 API Anahtarı Ayarlama:

Eğer gerçek AI yanıtları almak istiyorsanız:

  1. OpenRouter.ai sitesinden ücretsiz API anahtarı alın
  2. chatbot.js dosyasındaki YOUR_API_KEY_HERE kısmını değiştirin
  3. Sayfayı yenileyin
const API_KEY = 'sk-or-v1-sizin-api-anahtariniz-buraya';
